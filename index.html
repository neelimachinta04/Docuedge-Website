DocuEdge Capture

DocuEdge Capture consist of a modern web-based scanning interface along with the set of microservices leveraging state-of-the-art document classification and data extraction technologies. Capture ingests documents from multiple sources to perform image pre-processing, classification, and indexing(manual and automated).

Capture supports User Interface scanning, OCR, and Indexing, and Auto Document Classification.

Capture User Interface

Capture has a web based scanning interface, can enhance image quality with added features like zoom, rotate, split, re-order. It allows role based access control with an HTML 5 based interface that can be embedded in any web application.

Classification and Extraction

Capture reads and interprets machine encoded data to extract required data to support line of business activities and storage. It can recognize and classify documents. Capture can integrate with leading class natural language processing (NLP) classification services such as AWS Comprehend, Google Vision, Azure Document Understanding.

Validation and Post Processing

Capture converts scanned images into required output format (e.g. PDF) and ingests files and associated metadata in Docuedge Vault and link it with the LOB applications

DocuEdge Vault

Vault is a lightweight cloud native document repository. The solution is built on API first approach, which enables integration with any line of business application. It hosts documents and their related metadata securely in the form of cloud storage and offers tiered storage with retention rules. It leverages Content Services based model to provide content management and metadata management capabilities.

Key features of Vault include document library services, metadata and index-based searching, document workflows, with retention and hold management capabilities.

Document Ingestion

For storage, data needs to be extracted from source system into Vault. This process of loading data from a variety of sources is done using: 1. REST APIs to upload documents 2. Migration utility for bulk document upload.

Storage and Management

Documents in vault are stored in nested document categories. Metadata is defind for each category an access privileges are defined so that authenticated users have access to the data. Vault supports all file types and there is no file size limit. All library services such as versioning, file locking are also available.

Retention and Hold

For legal and compliance purposes, retention rule can be configured and applied for the data stored in Vault. The legal holds can be defined and applied. For data that needs to be disposed, purge eligible approval prior to disposition is also available. Vault also allows auto disposition of both metadata and documents once they are ready to be purged.

Search and Retrieval

Once documents have been stored, users can retrieve the data when needed using configurable search templates for metadata based searches on Vault. It also allows ful text search including searching in image files.

DocuEdge Archive

Archive is DocuEdge's information archiving and compliance repository which offers a single repository to archive structured and unstructured content from multiple applications.

It enables record level storage retrieval and compliance management with the following features :

Information Ingestion

Archive supports both structured and unstructured data and any data format or structure can be archived. It works with any ETL software.

Retention and Hold

For legal and compliance purposes, if there are retention rules to be applied for data archived, these can be configured. The retention rule set is configurable and legal holds can be derived and applied. For data that needs to be disposed, purge eligible approval prior to disposition is also available. Archive allows auto disposition of both metadata and documents once they are ready to be purged.

Storage and Retrieval

Archive supports multiple repositories under one configuration to keep application data separate. Configurable custom search templated allow easy access to data. In addition, Archive integrates with external reporting or data access tools.
